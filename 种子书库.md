# "书方"种子书库（30 本）

> 使用说明：以下 30 本书覆盖全部 8 个情绪标签，每个标签至少 3 本主推书。
> 选书原则：
> 1. 微信读书可获取，或网上容易找到电子版
> 2. 篇幅适中，不设阅读壁垒（最长不超过 20 万字，"瓦尔登湖"例外但仍推荐）
> 3. 具备"认知重塑"空间——读完会重新看待自己的情绪，而非仅仅被安慰
> 4. 体裁多元：小说/散文/书信/对话体/心理学/笔记小品/哲学，避免单一口感
> 5. 国籍/时代跨度：中国当代、中国现代、中国古典、日本、欧洲、美国、俄罗斯
>
> 优先级：P0 = 首批必填完整案例（10 本），P1 = 第二批补齐（10 本），P2 = 第三批（10 本）

## 二、按标签索引

> 格式：书名 — 一句话匹配理由。主标签书排前面。

### 迷茫（4 本主推）
| 书名 | 匹配逻辑 |
|------|----------|
| 棋王 | 王一生"把命放在棋里"——迷茫的解药不是规划，是找到一件事然后做到底 |
| 德米安 | 一个少年在善恶之间找到"真正的自我"，黑塞写的就是每个大学生的内心传记 |
| 给青年的十二封信 | 朱光潜用极克制的笔调谈读书、谈动、谈静、谈人生——像一位不会说教的长辈 |
| 山月记 | "我深怕自己本非美玉，故而不敢加以刻苦琢磨"——中岛敦把自我怀疑写成了一只变成老虎的诗人 |

### 浮躁（3 本主推）
| 书名 | 匹配逻辑 |
|------|----------|
| 心流 | 用认知科学告诉你：专注不是苦行，是最高级的快乐 |
| 陶庵梦忆 | 一个经历过极致繁华后又失去一切的人，回望时反而字字冷静 |
| 箭术与禅心 | 拉弓、放箭、命中——一个德国哲学家在学箭中悟出的"不费力"的专注 |

### 无聊（4 本主推）
| 书名 | 匹配逻辑 |
|------|----------|
| 万物有灵且美 | 一个乡下兽医的日常——母牛难产、狗儿撒娇——无聊是因为你没凑近了看 |
| 素履之往 | 木心一句话够你想一整天，"无聊"和"独处"之间只差一层认知的窗户纸 |
| 旅行的艺术 | 阿兰·德波顿教你在家门口旅行——出发的冲动不是逃离此地，是重新发现此地的愿望 |
| 查令十字街 84 号 | 一个纽约穷作家和一个伦敦旧书店之间通了二十年信——两个从未见面的人，靠对书的共同热爱，把平淡通信活成了一种"比见面更亲密"的关系 |

### 平淡（4 本主推）

## 五、下一步操作

1. **P2 优先（10 本）**：按 `样书案例模板.md` 的格式，逐本填写完整的认知重塑导读 + 金句采集
2. **金句采集方法**：每本书去微信读书"热门划线"区取前 5 条，筛选最能"一针扎醒"的 3-5 条
3. **导读写作**：可先手写 3-5 本作为风格锚点，剩余用 Gemini 3.1 Pro 的 Prompt 批量生成后人工校对
4. **标签冲突检查**：填完后回查——同一标签下的书，导读角度是否互斥/互补，避免同质化
