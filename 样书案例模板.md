# "书方"样书案例 —— 给另一个 AI 看的完整示例

> 使用说明：下面以"棋王"为例，完整演示从"用户情绪"到"最终呈现"的全链路。
> 当前书库规模：30 本，8 标签，每标签 3-4 本主推书。
> 后续每本入库的书都按此格式填写。

---

## ⚠️ 关键 UI 规则（开发者强制遵守）

> 以下规则源自 UI 审查，是"书方"区别于普通书评网站的核心差异点。**任何情况下不得违反。**

### 规则 1：标签点击 → 直达处方（无中间页）
- 用户点击情绪标签后，直接进入单本书的认知重塑导读页。
- **不展示多本书的列表让用户二次选择。** 药剂师不会给你一排药让你自己挑。
- 不想看这本？底部"换一剂"随机换一本同标签的书。

### 规则 2：书名后置（导读区不出现书名）
- 处方页（认知重塑导读区）**不显示书名、作者、封面**。
- 书名和作者**首次出现**在金句区的开头。
- 原因：先用药，后知药名。用户因情绪而来，不因书名而来。书名过早出现会产生"哦这本我听过/没听过"的预判，破坏导读的认知重塑效果。

### 规则 3：金句逐条淡入（非一次性展示）
- 3-5 条金句**逐条出现**，用户每下滑一个屏幕高度触发一条新金句。
- 使用 IntersectionObserver + opacity 0→1 淡入动画（duration 0.6s, easing: ease，无位移、无缩放）。
- 金句之间用 1px 分隔线（color: #e8e6e3），间距为导读段落间距的 2 倍。

### 规则 4：↓ 箭头固定在视口底部
- 处方页底部 ↓ 箭头使用 `position: fixed; bottom: 24px`。
- 防止长导读（>300 字）将箭头挤出首屏。
- 带轻微呼吸动画（opacity: 0.1 → 0.3, 2s 循环）。

### 规则 5：视觉规范
```
背景色       #fafaf9（暖白，非纯白）
文字色       #1a1a1a（非纯黑）
字体         导读/金句 → 衬线体 (font-family: "PingFang SC", "Noto Serif SC", "Source Han Serif SC", serif)
             情绪标签  → 无衬线 (font-family: "PingFang SC", "Noto Sans SC", sans-serif)
             视觉区分"选择模式"和"阅读模式"
字号         导读 16px / 行高 2.0 ｜ 金句 18px / 行高 2.2
页面最大宽度  640px，居中
左右内边距    32px（移动端）/ 48px（≥640px）
动画         仅页面切换淡入 (opacity 0→1, 400ms) + 箭头呼吸 + 金句逐条淡入
             不用滑动、弹跳、缩放、渐变背景
```

### 规则 6：金句区终止符
- 最后一条金句之后，渲染一个极小的空心方框 `□`（font-size: 10px, color: #ccc）。
- 古典印刷品的终章标记，暗示"此药已服完"。

## 🩺 认知重塑导读（"处方"核心）

你并不是不知道自己要什么。你知道得太多了。

大学像一张摆满菜单的桌子，所有人都在告诉你"这个有前途""那个最保值"，你每样都尝了一口，却没有一样真正饿了你的胃。迷茫不是选项不够多的病，恰好相反——是你脑子里塞了太多别人的菜单，反而听不见自己胃在叫。王一生面对这个问题的时候，他的解法近乎粗暴：别想了，下棋。别人串联、刷题、走关系，他就坐在那儿，跟一盘棋较劲。他把命放在棋里，棋就还他一个不被时代冲走的位置。

这个叫"棋呆子"的人身上有一种你不太能在校园里看到的品质：他不急于解释自己。别人觉得他呆、觉得他怪，他就笑一笑，继续摆棋。不争辩、不剖白、不写小作文自我辩护。这种近乎沉默的笃定，在一个人人都在经营人设的环境里，是一种很稀缺的定力。迷茫的时候，你最需要的不是更清晰的规划，而是先找到一件不需要向任何人解释理由的事情，然后先把它做完。

读完《棋王》大约需要一个下午。不长，刚好够你从宿舍床铺滑到书桌前。不用做笔记，不用画思维导图，你就看王一生怎么在火车上吃饭——米饭粒儿沾了一脸，一粒一粒捡起来吃掉——然后你会知道，"活着"这事儿，有时候就藏在把一件小事做到底的执拗里。

## 💊 金句试读区（下滑后展示）

**第 1 剂**：

> 我因为不喜欢象棋，也就不去关心什么象棋冠军，但王一生的大名，却常被班上几个棋篓子供在嘴上，我也就对其事迹略闻一二，知道王一生外号棋呆子，棋下得很神不用说，而且在他们学校那一年级里数理成绩总是前数名。我想棋下得好而有个数学脑子，这很合情理，可我又不信人们说的那些王一生的呆事，觉得不过是大家寻佚闻鄙事以快言论罢了。

**第 2 剂**：

> 棋开始了。上千人不再出声儿。只有自愿服务的人一会儿紧一会儿慢地用话传出棋步，外边儿自愿服务的人就变动着棋子儿。风吹得八张大纸哗哗地响，棋子儿荡来荡去。太阳斜斜地照在一切上，烧得耀眼。前几十排的人都坐下了，仰起来看，后面的人也挤得紧紧的，一个个土眉土眼，头发长长短短吹得飘，再没人动一下，似乎都要把命放在棋里搏。
>
> 我心里忽然有一种很古的东西涌上来，喉咙紧紧地往上走。读过的书，有的近了，有的远了，模糊了。平时十分佩服的项羽、刘邦都在目瞪口呆，倒是尸横遍野的那些黑脸士兵，从地下爬起来，哑了喉咙，慢慢移动。一个樵夫，提了斧在野唱。忽然又仿佛见了呆子的母亲，用一双弱手一张一张地折书页。

**第 3 剂**：

> 不久，脚卵抬起头，看着王一生说："天下是你的。"抽出一支烟给王一生，又说："你的棋是跟谁学的？"王一生也看着脚卵，说："跟天下人。"

## 🤖 AI Prompt 参考（给开发者的备注）

> 如果后续用 AI 批量生成导读，将以下 prompt 喂给大模型。变量用 {{}} 标注。

```
你是一个名为"书方"的精神向导。你的任务是为一本具体的书撰写"认知重塑导读"。

## 你的定位
- 你不是书评人，你是精神引路人。
- 你不提供廉价共情，你提供高一个维度的认知视角。
- 你的读者是深陷碎片化信息、注意力涣散、情绪漂浮但内心渴望深度阅读的大学生。

## 你的写作要求
- 200-400 字，三个自然段。
- 第一段：直击读者当下的情绪状态，用认知重塑的视角重新定义这种情绪。
- 第二段：引入这本书的核心洞见，说明它为什么恰好能回应这个状态。
- 第三段：给一个具体的、可执行的阅读入口（不要宏大叙事，要像朋友递来一杯水）。
- 文风：克制、锐利、有智识密度。不用感叹号。不写"你一定会有收获"这种废话。

## 这本书的信息
- 书名：{{书名}}
- 作者：{{作者}}
- 一句话简介：{{简介}}
- 匹配的情绪标签：{{标签}}
- 代表性金句：
{{金句列表}}

现在请为这本书撰写认知重塑导读。
```
